# CampusLib - Library Management System

![CampusLib Logo](assets/images/logo.png) *(Placeholder for logo image)*

**CampusLib** is a modern library management system designed for university libraries to streamline book borrowing, returning, user management, and fine handling. It features a Flutter-based mobile app for cross-platform support (Android and iOS) and a Flask-based REST API backend with a MySQL database. The system supports two user roles: **Readers** (students/faculty) and **Admins** (librarians), with QR code-based workflows for borrowing and returning books.
